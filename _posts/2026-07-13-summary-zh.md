## [Chromium 148：Math.tanh 通过浮点差异可识别操作系统](https://scrapfly.dev/posts/browser-math-os-fingerprint/) ⭐️ 8.0/10

自 Chromium 148 起，Math.tanh 函数暴露了操作系统级别的浮点舍入差异，网站可通过测量单次函数调用的精度变化来识别底层操作系统。 这一发现引入了一种难以伪造的新型浏览器指纹识别向量，可将浏览器行为与操作系统关联，引发重大隐私担忧，并可能使反机器人系统更加精准。 该技术利用了 Math.tanh、CSS 三角函数和 Web Audio 压缩器都通过宿主 libm 库实现的特点，因此即使伪造了 User-Agent，舍入差异也会暴露真实操作系统。

hackernews · joahnn_s · 7月12日 21:12 · [社区讨论](https://news.ycombinator.com/item?id=48884853)

**背景**: 浏览器指纹识别通过聚合屏幕尺寸、字体和 API 行为等设备属性来识别用户，无需使用 Cookie。由于不同操作系统使用不同的数学库（libm），浮点运算会产生细微但可检测的差异。Chromium 148 更新后的 Math.tanh 实现使这些操作系统特定的舍入行为可通过 JavaScript 访问，从而催生了一种新的指纹识别方法。
